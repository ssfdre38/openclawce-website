# Introducing OpenClaw Community Edition

We're excited to announce **OpenClaw Community Edition** - a community-driven fork of OpenClaw that puts control back in the hands of users.

## Why a Community Edition?

OpenClaw is an incredible multi-channel AI gateway with powerful features for managing AI conversations across platforms. However, as with many evolving projects, the direction and priorities of the original project may not always align with what every user needs.

That's why we created **OpenClaw Community Edition**.

## What Makes CE Different?

### 🎯 Community-Driven Development
- Feature requests and bug fixes driven by actual user needs
- Transparent development process on GitHub
- Open to contributions from anyone

### 🔓 True Open Source Spirit
- No proprietary extensions or closed features
- Clear, permissive licensing (MIT)
- Full access to all source code

### 🛠️ Self-Hosting Friendly
- Optimized for self-hosting scenarios
- Clear documentation for deployment
- No "phone home" features or telemetry without consent

### ⚡ Performance First
- Focus on efficiency and speed
- Removed unnecessary overhead
- Better resource utilization for local deployments

## Current State

OpenClaw CE is based on OpenClaw v2026.3.2 with several important improvements:

- **Fixed critical bugs** including Ollama tool format issues
- **Enhanced security** with improved gateway authentication
- **Better configuration** through environment variables
- **Centralized metadata** for easier customization
- **Community branding** that respects the original project

## Our Mission

We believe powerful AI tools should be:
- **Accessible** - Easy to set up and use
- **Transparent** - No hidden features or behaviors  
- **Customizable** - Adaptable to your specific needs
- **Community-owned** - Shaped by the people who use it
